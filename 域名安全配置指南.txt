二、DNSSEC 配置
================================================================================

📋 状态：需要在域名注册商处配置

DNSSEC（Domain Name System Security Extensions）可以防止 DNS 缓存投毒攻击。

配置步骤：

1. 登录您的域名注册商（例如：GoDaddy、Namecheap、阿里云等）
2. 找到 goldiandesk.top 域名的 DNS 管理页面
3. 查找 "DNSSEC" 或 "域名系统安全扩展" 选项
4. 启用 DNSSEC 功能
5. 配置 DNSSEC 密钥（通常由注册商自动生成）

常见域名注册商的 DNSSEC 配置：

- **GoDaddy**:
  1. 登录 GoDaddy 账户
  2. 进入 "我的产品" → "域名"
  3. 选择 goldiandesk.top
  4. 点击 "DNS" → "DNSSEC"
  5. 点击 "启用 DNSSEC"

- **Namecheap**:
  1. 登录 Namecheap 账户
  2. 进入 "域名列表"
  3. 选择 goldiandesk.top
  4. 点击 "高级 DNS"
  5. 找到 "DNSSEC" 部分并启用

- **阿里云**:
  1. 登录阿里云控制台
  2. 进入 "域名" 服务
  3. 选择 goldiandesk.top
  4. 点击 "DNS 设置" → "DNSSEC"
  5. 启用 DNSSEC

验证方式：
使用在线工具验证 DNSSEC 是否启用：
- https://dnssec-analyzer.verisignlabs.com/
- 输入 goldiandesk.top 进行检查

三、WAF（Web 应用防火墙）配置
================================================================================

📋 状态：需要在 Netlify 或 Cloudflare 配置

选项1：使用 Netlify 的安全功能（推荐）

Netlify 提供基础的安全保护：
1. 登录 Netlify Dashboard
2. 进入您的网站设置
3. 找到 "Security" 或 "安全" 选项
4. 启用以下功能：
   - Bot protection（机器人防护）
   - DDoS protection（DDoS 防护）
   - Rate limiting（速率限制）

选项2：使用 Cloudflare（更强大）

如果需要更高级的 WAF 功能，可以使用 Cloudflare：

1. 注册 Cloudflare 账户（免费版即可）
2. 添加 goldiandesk.top 到 Cloudflare
3. 按照 Cloudflare 的指引更改域名的 DNS 服务器
4. 在 Cloudflare 中启用：
   - WAF（Web Application Firewall）
   - Bot Management
   - DDoS Protection
   - Rate Limiting
   - SSL/TLS（设置为 "Full" 或 "Strict"）

Cloudflare WAF 规则配置建议：
- 启用 "Cloudflare Managed Ruleset"
- 启用 "OWASP Core Ruleset"
- 设置适当的安全级别（Medium 或 High）

五、FortiGuard 拦截解决方案
================================================================================

当前问题：goldiandesk.top 被 FortiGuard 拦截，原因是"新注册的域名"

解决方案：

方案1：请求重新评估（最快）
1. 访问 FortiGuard 的网站评级页面：
   https://www.fortiguard.com/webfilter
2. 在搜索框中输入：goldiandesk.top
3. 点击 "Lookup" 或 "查询"
4. 查看当前评级
5. 如果评级过低，点击 "Request a Review" 或 "请求重新评估"
6. 填写表单并提交
7. 通常 24-48 小时内会有结果

方案2：等待自动更新
- FortiGuard 通常会在 1-3 天内自动更新新域名的评级
- 在此期间，可以使用 Netlify 的默认域名访问

方案3：使用 VPN
- 临时使用 VPN 绕过 FortiGuard 拦截
- 仅用于测试，不适合长期使用

六、Netlify 默认域名
================================================================================

如果 goldiandesk.top 暂时无法访问，可以使用 Netlify 的默认域名：

查找默认域名的步骤：
1. 登录 Netlify Dashboard
2. 进入您的网站
3. 在 "Site overview" 或 "网站概览" 页面
4. 找到 "Site settings" 或 "网站设置"
5. 点击 "Domain management" 或 "域名管理"
6. 您会看到 Netlify 分配的默认域名，格式类似：
   https://your-site-name.netlify.app

使用默认域名的好处：
- 不会被 FortiGuard 拦截
- 立即可用
- 同样支持 HTTPS
